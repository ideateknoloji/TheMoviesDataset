# TheMoviesDataset
The Movies Dataset with Turkish Column Headers

Bu depoda (repository) yaklaşık 20000 film verisi aşağıdaki bilgilerle birlikte bulunmaktadır.

Bilgi alanları:
  - film adı
  - orijinal adı
  - vizyon tarihi
  - yönetmen
  - oyuncular
  - özet ve detay
  - dil
  - bütçe
  - yapım yılı
  - renk
