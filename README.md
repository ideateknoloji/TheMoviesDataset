# TheMoviesDataset
The Movies Dataset with Column Headers in Turkish

### English 

In this repository, the movies dataset includes nearly 20000 data with data fields noted below.

Fields:
  - film adı        
  - orijinal adı    
  - vizyon tarihi    
  - yönetmen
  - oyuncular
  - özet ve detay
  - dil
  - bütçe
  - yapım yılı
  - renk
